# Proyecto-Hackaton-ONE-Equipo46-ChurnInsight
El desafío de ChurnInsight es crear una solución que prediga si un cliente es propenso a cancelar un servicio.  El equipo de Data Science desarrollara un modelo predictivo y el equipo de Back-end construira una API para disponibilizar esa predicción a otros sistemas, para que el negocio actúe antes de que el cliente decida irse.
